## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/WebDev.git
   cd WebDev
   ```

2. **Navigate to specific topics**
   ```bash
   cd HTML-CSS/basics  # For HTML/CSS fundamentals
   cd JavaScript/fundamentals  # For JavaScript basics
   ```

3. **Install dependencies** (for projects with package.json)
   ```bash
   npm install
   ```

4. **Run examples**
   ```bash
   # For static files
   open index.html
   
   # For Node.js projects
   npm start
   ```
